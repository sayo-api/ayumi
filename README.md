## COMO INSTALAR NO TERMUX
```bash
> termux-setup-storage
> pkg update && pkg upgrade
> pkg install git
> pkg install nodejs
> pkg install bash
> pkg install ffmpeg
> pkg install libwebp
> git clone https://github.com/sayo-api/ayumi
> cd ayumi
> npm i
> node run.js
# para o sdcard (arquivo baixado)
> cd /sdcard
> cp -r ayumi $HOME
> cd ayumi
> npm i
> npm start
```
